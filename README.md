# 🏘️ Grão Bairro Fortaleza

**Protótipo interativo de inteligência de bairros para microempreendedores**

Projeto acadêmico — UNIFOR 2026.1 — Atividade Final: Ideações para Soluções Tecnológicas

**ODS 8:** Trabalho Decente e Crescimento Econômico

---

## 🎯 O que é

Grão Bairro transforma dados públicos (RAIS, IBGE, CAGED, Brasil API) em inteligência prática para quem quer empreender no bairro certo.

## 🛠️ Ferramentas

| Ferramenta | Descrição |
|-----------|-----------|
| **📍 Onde Abrir?** | Recomendação do melhor bairro por setor de negócio |
| **🌡️ Termômetro** | Indicadores econômicos em tempo real por bairro |
| **🏪 Vitrine** | Mapa de empresas ativas em cada bairro |
| **💰 Contracheque** | Poder de compra e potencial de consumo |
| **🎯 Radar MEI** | Nichos subatendidos e oportunidades de formalização |

## 📊 Fontes de dados

- **RAIS 2024** — 1.423.678 vínculos formais em Fortaleza
- **IBGE** — Censo 2022 (população, domicílios, setores censitários)
- **CAGED** — Movimentação de emprego (admissões/desligamentos)
- **Brasil API** — Dados de CNPJ
- **POF 2017-2018** — Padrões de consumo por faixa de renda
